AI Hearing System

Tech Stack: Python, Vosk, NLP Summarization, gTTS, Speech Processing

Developed an AI-powered hearing assistance system to facilitate communication for individuals with speech and hearing impairments.
Integrated the Vosk Speech Recognition Model for real-time speech-to-text conversion by processing audio streams into text chunks.
Implemented live captioning to display spoken content instantly, improving accessibility during conversations.
Applied text summarization techniques to generate concise summaries of lengthy conversations and spoken content.
Utilized Google Text-to-Speech (gTTS) for converting generated text responses into natural-sounding speech.
Designed a real-time processing pipeline for speech recognition, text summarization, caption generation, and voice output.
Enhanced communication accessibility by providing both textual and audio-based interaction support.
